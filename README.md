# Permission Saver

Кроссплатформенное приложение для сохранения и восстановления настроек доступа (ACL) к файлам и директориям.

## ⚠️ ВАЖНО: Запуск от имени администратора

Для корректной работы приложения **обязательно** запускать его с повышенными привилегиями.

### Как запустить:

**Windows**
1. Найдите файл `PermissionSaver.exe` (или ярлык).
2. Нажмите **правой кнопкой мыши**.
3. Выберите **"Запуск от имени администратора"**.
   *(Или запустите PowerShell от имени администратора и введите `.\PermissionSaver.exe`)*.

**🐧 Linux / 🍎 macOS**
1. Как-то, у меня нет Linux/macOS

## 🚀 Использование

### 1. Сохранение (Сериализация)
1. В блоке **"Сериализация"** укажите путь к файлу или папке.
2. Укажите путь, куда сохранить `.json` архив.
3. Нажмите **"💾 Сохранить файл и права"**.
   * *Результат:* Создается JSON-файл, содержащий содержимое (Base64) и полные права доступа.

### 2. Восстановление (Десериализация)
1. В блоке **"Десериализация"** выберите созданный ранее `.json` файл.
2. Укажите папку назначения.
3. Нажмите **"📂 Восстановить файл и права"**.
   * *Результат:* Файлы создаются заново с идентичными правами доступа.

## 📦 Формат данных
Приложение упаковывает всё в один **JSON** файл:
*   **Содержимое:** Кодируется в Base64.
*   **Windows:** Сохраняет строку SDDL (Security Descriptor Definition Language).
*   **Linux/macOS:** Сохраняет числовой режим доступа (UnixFileMode).
